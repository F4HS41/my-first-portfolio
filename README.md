## 🚀 โครงสร้างเว็บไซต์ในโปรเจกต์นี้
* **index.html :** หน้าแรกของเว็บไซต์ นำเสนอประวัติส่วนตัว แรงบันดาลใจ และช่องทางการติดต่อ
* **projects.html :** หน้าคลังผลงาน รวบรวมโปรเจกต์ที่เคยทำ ภาพถ่ายเกียรติบัตร และเหตุผลเบื้องหลังความสำเร็จ
* **style.css :** ไฟล์ศูนย์กลางที่ใช้ควบคุมหน้าตาและธีมสีของเว็บไซต์ทั้งหมด
